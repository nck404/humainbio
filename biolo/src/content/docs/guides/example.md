# 1. Hệ vận động (cơ – xương – khớp)
**Gồm:**
- Xương: xương đùi, xương cánh tay, xương sọ, xương sườn…
- Khớp: khớp gối, khớp vai, khớp cổ tay…
- Cơ: cơ tay, cơ chân, cơ bụng, cơ liên sườn…

# 2. Hệ tuần hoàn (tim – máu – mạch máu)
**Gồm:**
- Tim  
- Mạch máu: động mạch, tĩnh mạch, mao mạch  
- Máu: hồng cầu, bạch cầu, tiểu cầu, huyết tương

# 3. Hệ hô hấp
**Gồm:** Mũi, Họng, Khí quản, Phế quản, Phổi, Phế nang

# 4. Hệ tiêu hóa
**Gồm:**
- Miệng: lưỡi, răng, tuyến nước bọt  
- Thực quản  
- Dạ dày  
- Ruột non  
- Ruột già  
- Trực tràng – hậu môn  
- Tuyến tiêu hoá phụ: gan, túi mật, tụy

# 5. Hệ bài tiết – thải độc
**Gồm:**
- Thận  
- Niệu quản  
- Bàng quang  
- Niệu đạo  
- Da (tuyến mồ hôi)  
- Phổi (thải CO₂)

# 6. Hệ thần kinh
**Gồm:**
- Não bộ: đại não, tiểu não, hành não  
- Tủy sống  
- Dây thần kinh  
- Cơ quan cảm giác: mắt, tai, mũi, lưỡi, da

# 7. Hệ nội tiết (hormone)
**Gồm:**
- Tuyến yên  
- Tuyến giáp  
- Tuyến cận giáp  
- Tuyến thượng thận  
- Tuyến tụy nội tiết  
- Tuyến sinh dục (buồng trứng / tinh hoàn)

# 8. Hệ sinh dục
**Nam:** Tinh hoàn, Mào tinh, Ống dẫn tinh, Tuyến tiền liệt, Dương vật  
**Nữ:** Buồng trứng, Vòi trứng, Tử cung, Âm đạo, Tuyến vú

# 9. Hệ miễn dịch
**Gồm:** Hạch bạch huyết, Lách, Tuyến ức, Bạch cầu, Tuỷ xương
